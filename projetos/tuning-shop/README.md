# Tuning Parts

Aplicativo Expo/React Native para uma loja de pecas automotivas de tuning.

## Rodar

```bash
npm install
npm run web
```

## Perfis de teste

- Admin: `admin@tuning.com` / `Admin123!`
- Usuario: `cliente@tuning.com` / `Cliente123!`

Quando receber a conexao real do Firebase, substitua os valores em `src/services/firebase.ts`.
